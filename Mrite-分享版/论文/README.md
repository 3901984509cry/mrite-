## A092核心格式特征

### 摘要句式（最具特色）

```latex
\textbf{针对问题X，}我们构建了【模型名称】。
首先，【步骤1】；
接着，【步骤2】；
然后，【步骤3】；
最后，【步骤4】。
最终我们得到了【指标1】为\textbf{【数值】}，
【指标2】为\textbf{【数值】}，其余结果详见表X和表X。
```

### 符号说明（4列格式，区别于原2列模板）

```latex
\begin{longtable}{
  >{\centering\arraybackslash}p{0.10\textwidth}
  >{\centering\arraybackslash}p{0.36\textwidth}
  >{\centering\arraybackslash}p{0.10\textwidth}
  >{\centering\arraybackslash}p{0.36\textwidth}}
  \caption{符号说明} \\
  \toprule
  符号 & 说明 & 符号 & 单位 \\
```

### 模型假设（enumerate，不加粗编号）

```latex
\begin{enumerate}
\item 假设【内容】
\item 假设【内容】
\end{enumerate}
```

### 模型检验（嵌入在每问求解末尾）

每问最后加 `\paragraph{模型检验}`，随机生成100组可行解与最优解比较。

### 参考文献（极简风格，3条）

```latex
\begin{thebibliography}{99}
\bibitem{ref1} 作者. 标题[D]. 大学, 年份.
\bibitem{ref2} 作者. 标题[D]. 大学, 年份.
\bibitem{ref3} 作者1, 作者2等. 标题[J]. 期刊, 年份, 卷(期):页码.
\end{thebibliography}
```

## 与原模板的差异对比

| 维度 | 原模板 | 新模板（A092风格） |
|------|--------|-----------------|
| 符号说明 | 2列（符号\|说明） | **4列（符号\|说明\|符号\|单位）** |
| 模型假设 | itemize + \textbf{假设N：} | **enumerate，不加粗** |
| 摘要句式 | "对于问题X，首先...接着..." | **"针对问题X，我们构建了XX模型。首先...；接着...；然后...；最后...。最终..."** |
| 问题分析 | 无独立章节 | **独立第四章** |
| 模型检验 | 独立第6章 | **嵌入在每问求解末尾** |
| 参考文献数量 | 8-15条 | **3-5条（简洁风格）** |
| 附录内容 | 简单文件清单 | **文件清单+完整代码（原文20页代码）** |
