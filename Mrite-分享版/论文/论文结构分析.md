## 1. 章节层次结构

```
摘要（第1页）

    1.1 问题背景
    1.2 问题重述（逐问：问题一/二/三）
二、模型假设（第2页）
    4条编号假设
三、符号说明（第3页）
    4列表格（符号|说明|符号|单位）
四、问题分析（第3-4页）
    4.1 问题一的分析
    4.2 问题二的分析
    4.3 问题三的分析
五、模型的建立与求解（第4-32页）
    5.1 问题一模型的建立与求解
        5.1.1 定日镜场年平均输出热功率模型
            （1）太阳的高度角和方位角
            （2）定日镜的俯仰角和方位角
            （3）光学效率
            （4）年均输出热功率
        5.1.2 求解结果
    5.2 问题二模型的建立与求解
        5.2.1 单目标优化模型
        5.2.2 遗传算法求解
        5.2.3 模型检验
    5.3 问题三模型的建立与求解
        5.3.1 非统一优化模型
        5.3.2 求解方法
        5.3.3 模型检验（含三问对比）

附录（第34-53页）
    附录1：文件清单
    附录2-N：MATLAB代码全文
```

**注意**：原论文无独立的"模型评价"和"模型改进推广"章节，模型检验嵌入在每问求解末尾。

## 2. 格式特征

### 封面信息
- 论文正文第1页顶端直接开始，无独立封面页
- 标题居中，一级标题风格

### 摘要格式
- 每问格式：**"针对问题X，我们构建了XX模型。首先...；接着...；然后...；最后...。最终我们得到了【数值结果】，其余结果详见表X和表X。"**
- 关键数值加粗
- 结尾一句话总结
- 无关键词列表（但标准模板需要）

### 问题重述格式
- 1.1 问题背景：2-3段，从宏观到具体
- 1.2 问题重述：每问一段，加粗"问题X："，内容为问题的精炼表述

### 模型假设格式
- 使用 `enumerate` 编号列表（而非 `itemize`）
- 格式："假设..."，不加粗编号

### 符号说明格式
- **4列表格**（不同于原模板的2列）：符号 | 说明 | 符号 | 单位
- 每行放两个符号及其说明

### 问题分析格式（独立章节）
- 每问一个subsection
- 每问格式：先说"问题X要求我们..."（1句），然后"首先...；接着...；然后...；最后..."（步骤串联），末尾"因此我们可以采用..."
- 无流程图（流程图在建模章节）

### 模型建立与求解格式
- 每问一个subsection（5.X）
- 子小节用"（1）标题"格式（圆括号编号）
- 每个分析阶段先有流程图（图X 问题X流程图）
- 公式用 equation 环境，编号在右侧
- 推导过程中夹叙夹议（不孤立堆砌公式）
- 变量说明紧跟公式后

### 模型检验格式
- 嵌入在每问求解末尾（不是独立章节）
- 方法：随机生成100组可行解，与最优解比较
- 展示方式：针状图/柱状图

### 参考文献格式
- 手动 thebibliography 环境
- 编号格式：[1][2][3]...
- 条目格式：作者. 标题[类型]. 出版社/期刊, 年份. 或 年份,卷(期):页码.
- 数量：仅3条（简洁风格）

### 附录格式
- 先列文件清单（文件名+说明）
- 再逐个贴出 MATLAB 代码全文
- 代码量很大（占约20页）

### 图表特征
- 图片宽度：未明确指定，推测约0.7-0.8\textwidth
- 图片标题：在图片下方，"图X 标题"格式
- 表格：无明显三线表，使用基本tabular
- 坐标轴标签：全中文

## 4. 与原模板（论文/）的主要差异

| 维度 | 原模板 | 新模板（A092风格） |
|------|--------|-----------------|
| 符号说明表格 | 2列（符号\|说明） | 4列（符号\|说明\|符号\|单位） |
| 模型假设列表 | itemize + \textbf{假设N：} | enumerate，不加粗 |
| 摘要句式 | "对于问题X，首先...接着...最后..." | "针对问题X，我们构建了XX模型。首先...；接着...；然后...；最后..." |
| 问题分析 | 无独立章节（合并入引言） | 独立章节（四、问题分析） |
| 模型检验 | 独立第6章 | 嵌入在每问求解末尾 |
| 参考文献数量 | 8-15条 | 3条（简洁风格） |
| 附录内容 | 简单文件清单 | 文件清单+完整代码 |
| 代码语言 | Python | MATLAB |
